Contexto: Este script es el prototipo conceptual de la Sesión 1. Establece la lógica central de normalización (kcal absolutas → proporciones) y el pipeline de reducción dimensional (87 ítems → macrocategorías). Fue superado por 02_feature_engineering.ipynb, que implementa el mapeo completo con los 87 ítems y los fixes de la Sesión 2. Se conserva como referencia de las decisiones de diseño originales.
Principio lógico central

¿Por qué proporciones y no kcal absolutas?

→
Comparar kcal absolutas entre países introduce sesgo de tamaño: 1.000 kcal de carne en un país de 2.000 kcal/día es una dieta hipercarnívora (50%). En un país de 4.000 kcal/día, es moderada (25%).
→
Al forzar que todas las proporciones sumen 1.0, normalizamos el espacio euclidiano. K-Means mide distancias reales de comportamiento dietario, no de volumen poblacional.
→
La reducción de 87 variables a 7 macrocategorías elimina la maldición de la dimensionalidad y hace que el espacio de clustering sea interpretable.
Pipeline implementado (6 pasos)

Flujo de transformación

1
Diccionario de mapeo: cada ítem FAO se asigna a una de las 7 macrocategorías. Lo que no está en el diccionario se descarta como ruido.
2
Ingesta estricta de RAM: dtypes explícitos (category para strings, int16 para años, float32 para valores). Solo se cargan las columnas necesarias.
3
Filtrado por elemento: solo se retienen filas con Element == 'Food supply (kcal/capita/day)'. Aplicación del mapeo y eliminación de NaN (ítems no mapeados).
4
Agrupación y pivote: groupby(['Area', 'Year', 'Macro_Grupo']) → suma de kcal → pivot para tener una columna por macrocategoría.
5
Normalización composicional: cada macrocategoría se divide por Total_DES_Kcal. Las 7 proporciones resultantes suman exactamente 1.0 por fila.
6
Persistencia y limpieza: export a parquet + del de dataframes intermedios + gc.collect().
Riesgo documentado — Fuga de categorías: si un ítem como "Pescado" no está en el diccionario de mapeo, el Total_DES_Kcal de países pesqueros se subestima y sus proporciones restantes se inflan artificialmente. Solución: antes de descartar NaN, verificar que los ítems no mapeados no superen 50 kcal/día históricamente (df_raw[df_raw['Macro_Grupo'].isna()]['Item'].value_counts()).
Nota sobre varianza cero: si un país reporta 0 en Tubérculos (común en Europa del Este), el fillna(0) es correcto — no aplicar imputación de medias. Un 0 real es una señal válida para el clustering.

"""Cálculo de Dieta por País: Normalización
Prototipo conceptual — Sesión 1 (02/04/2026)
Aguacate Team · TFM "La Dieta de un País"

Nota: este script es el precursor de 02_feature_engineering.ipynb.
El mapeo aquí es parcial (~15 ítems). El mapeo completo (87 ítems)
está en mapeo_fao_completo.py
"""

import pandas as pd
import numpy as np
import gc

# 1. DICCIONARIO DE MAPEO (prototipo parcial ~15 ítems)
# El mapeo completo (87 ítems) está en mapeo_fao_completo.py
mapeo_fao = {
    'Wheat and products':    'Cereales',
    'Rice and products':     'Cereales',
    'Maize and products':    'Cereales',
    'Bovine Meat':           'Carnes',
    'Pigmeat':              'Carnes',
    'Poultry Meat':          'Carnes',
    'Mutton & Goat Meat':    'Carnes',
    'Milk - Excluding Butter': 'Lacteos_Huevos',
    'Eggs':                 'Lacteos_Huevos',
    'Soyabean Oil':          'Aceites_Grasas',
    'Palm Oil':             'Aceites_Grasas',
    'Sugar cane':           'Azucares',
    'Sugar beet':           'Azucares',
    'Cassava and products':  'Tuberculos',
    'Potatoes and products': 'Tuberculos'
    # ... (completado en mapeo_fao_completo.py)
}

# 2. INGESTA ESTRICTA (RAM is Money)
dtypes_fao = {
    'Area':    'category',
    'Year':    'int16',
    'Item':    'category',
    'Element': 'category',
    'Value':   'float32'
}
# df_raw = pd.read_csv('fao_data.csv', usecols=dtypes_fao.keys(), dtype=dtypes_fao)

# 3. FILTRADO: solo Food supply (kcal/capita/day)
df_kcal = df_raw[df_raw['Element'] == 'Food supply (kcal/capita/day)'].copy()
df_kcal['Macro_Grupo'] = df_kcal['Item'].map(mapeo_fao)
df_kcal = df_kcal.dropna(subset=['Macro_Grupo'])

# 4. AGRUPACIÓN Y PIVOTE
df_grouped = df_kcal.groupby(
    ['Area', 'Year', 'Macro_Grupo'], observed=True
)['Value'].sum().reset_index()

df_vector = df_grouped.pivot(
    index=['Area', 'Year'],
    columns='Macro_Grupo',
    values='Value'
).fillna(0).astype('float32')

# 5. NORMALIZACIÓN COMPOSICIONAL (el cálculo core)
df_vector['Total_DES_Kcal'] = df_vector.sum(axis=1)
macro_cols = df_vector.columns.drop('Total_DES_Kcal')
for col in macro_cols:
    df_vector[f'%_{col}'] = (df_vector[col] / df_vector['Total_DES_Kcal']).astype('float32')

# Eliminar columnas absolutas (ya no necesarias para clustering)
df_vector = df_vector.drop(columns=macro_cols).reset_index()

# 6. PERSISTENCIA Y LIMPIEZA
df_vector.to_parquet('data/processed/01_diet_vector_base.parquet')
del df_raw, df_kcal, df_grouped, df_vector
gc.collect()

print("--> Parsing Jerárquico completado. Vectores normalizados.")
