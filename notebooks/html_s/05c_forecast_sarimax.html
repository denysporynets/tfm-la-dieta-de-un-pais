Motor C (v3): Forecast Dietario con SARIMAX¶

Aguacate Team · TFM La Dieta de un País

Estrategia¶

Tercera implementación del Motor C, basada en modelos ARIMA/SARIMAX. Con datos anuales no existe componente estacional intra-año, por lo que SARIMA se reduce a ARIMA(p,d,q). Se selecciona el orden óptimo por AIC para cada serie, con un espacio de búsqueda conservador acorde al tamaño muestral (13 observaciones).

Prophet (nb05) LightGBM Global (nb05b) SARIMAX (nb05c)
Modelos 210 locales 1 global 210 locales
Datos por modelo 13 obs ~2100 totales 13 obs
Selección orden changepoints lag features AIC (p≤2,d≤1,q≤1)
IC Bayesiano Quantile regression Analítico (sigma²)
Forecast Directo Recursivo Directo

Resultado esperado: SARIMAX es el punto medio entre Prophet (tendencia suave) y LightGBM Global (captura patrones cruzados). Documentado para comparación rigurosa de metodologías.

# ── SELECCIÓN DE ORDEN ARIMA POR AIC ─────────────────────────────
# Espacio de búsqueda conservador para series de 13 puntos:
# p ∈ {0,1,2}, d ∈ {0,1}, q ∈ {0,1} → 12 combinaciones
# NOTA: se usa RangeIndex (no DatetimeIndex) para evitar bug de statsmodels
# 0.14.x donde get_forecast(steps=N) falla con YS frequency.

def best_arima_order(series, p_range=(0,2), d_range=(0,1), q_range=(0,1)):
    """Selecciona orden ARIMA(p,d,q) por AIC mínimo."""
    best_aic = np.inf
    best_order = (1, 1, 0)
    for p, d, q in product(range(p_range[0], p_range[1]+1),
                            range(d_range[0], d_range[1]+1),
                            range(q_range[0], q_range[1]+1)):
        if p + d + q == 0:
            continue
        try:
            model = SARIMAX(series, order=(p, d, q),
                            trend="c", enforce_stationarity=False,
                            enforce_invertibility=False)
            res = model.fit(disp=False, maxiter=300)
            if res.aic < best_aic:
                best_aic = res.aic
                best_order = (p, d, q)
        except Exception:
            continue
    return best_order, best_aic


def forecast_sarimax(series_vals, years, country, macro, horizon_year):
    """Entrena SARIMAX con orden óptimo y genera forecast con IC 80%.
    Usa RangeIndex para compatibilidad con statsmodels >= 0.14.
    """
    # RangeIndex: evita el bug de get_forecast con DatetimeIndex YS
    ts = pd.Series(series_vals.astype(float))  # índice 0..12

    order, aic = best_arima_order(ts)

    model = SARIMAX(ts, order=order, trend="c",
                    enforce_stationarity=False, enforce_invertibility=False)
    res = model.fit(disp=False, maxiter=300)

    n_steps = int(horizon_year) - int(years[-1])
    nobs = len(ts)
    forecast_res = res.get_prediction(start=nobs, end=nobs + n_steps - 1)
    ci = forecast_res.conf_int(alpha=0.20)  # IC 80%

    fc_years = list(range(int(years[-1]) + 1, int(horizon_year) + 1))
    rows = []

    # Histórico
    for yr, val in zip(years, series_vals):
        rows.append({
            "Area": country, "variable": macro, "Year": int(yr),
            "yhat": float(val),
            "yhat_lower": float(val),
            "yhat_upper": float(val),
            "is_forecast": False
        })

    # Forecast
    pred_mean = forecast_res.predicted_mean.values
    ci_vals   = ci.values
    for i, yr in enumerate(fc_years):
        yhat  = float(np.clip(pred_mean[i], 0, 1))
        lower = float(np.clip(ci_vals[i, 0], 0, 1))
        upper = float(np.clip(ci_vals[i, 1], 0, 1))
        rows.append({
            "Area": country, "variable": macro, "Year": yr,
            "yhat": yhat, "yhat_lower": lower, "yhat_upper": upper,
            "is_forecast": True
        })

    return rows, order, aic

# ── WALK-FORWARD VALIDATION (train ≤2019, val 2020–2022) ─────────
wf_preds, wf_trues = [], []
order_log = []

for country in df["Area"].unique():
    sub = df[df["Area"] == country].sort_values("Year")

    for macro in MACROS:
        train_vals = sub[sub["Year"] <= 2019][macro].values.astype(float)
        val_vals   = sub[sub["Year"] >= 2020][macro].values.astype(float)

        try:
            ts = pd.Series(train_vals)  # RangeIndex
            order, _ = best_arima_order(ts)
            model = SARIMAX(ts, order=order, trend="c",
                            enforce_stationarity=False, enforce_invertibility=False)
            res = model.fit(disp=False, maxiter=300)
            nobs = len(ts)
            fc = res.get_prediction(start=nobs, end=nobs + 2)
            preds = np.clip(fc.predicted_mean.values, 0, 1)
            wf_preds.extend(preds)
            wf_trues.extend(val_vals)
            order_log.append(order)
        except Exception:
            wf_preds.extend([train_vals[-1]] * 3)
            wf_trues.extend(val_vals)

wf_preds = np.array(wf_preds)
wf_trues = np.array(wf_trues)

mae_wf  = mean_absolute_error(wf_trues, wf_preds)
r2_wf   = r2_score(wf_trues, wf_preds)
rmse_wf = np.sqrt(mean_squared_error(wf_trues, wf_preds))

    index=['Area', 'Year'],

pivot.columns.name = None

last_cluster = df.groupby('Area')['cluster_id'].last().reset_index()

last_cpi = df.groupby('Area')['Food_CPI'].last().reset_index()

row_sums = pivot[MACROS].sum(axis=1)
